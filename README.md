# Horofree
Parking meter for freelancers

## Run the application
You need to install Elm on your computer, follow the instructions here: https://guide.elm-lang.org/get_started.html

Then

```bash
$ npm install
```

Answer yes when prompt by `elm-package`

```
$ npm run dev
```
